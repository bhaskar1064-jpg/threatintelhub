Query Domain Intelligence

Query reported scores, SSL state, and classification history by sending a simple JSON GET request.

# GET /api/v1/threats/verify?domain=scam-portal.net
curl -X GET "https://api.threatintelhub.io/v1/threats/verify?domain=scam-portal.net" \
    -H "Authorization: Bearer YOUR_API_TOKEN"

Response Payload Example

{
  "domain": "scam-portal.net",
  "status": "Verified Malicious",
  "dangerScore": 92,
  "category": "Phishing",
  "sslEnabled": false,
  "whois": {
    "registrar": "NameCheap, Inc.",
    "registrationDate": "2026-06-15"
  }
}

About ThreatIntel Hub

A global community protecting users and networks from modern internet threats.

Community-Driven Vetting

Our platform pairs automated heuristics and registrar validation with crowd-sourced verification. Reports submitted by veteran analysts are weighted with high trust immediately, while reports from anonymous users undergo vetting by community moderators before classification shifts to Verified.

Platform Mandate

We believe in an open web. ThreatIntel Hub aggregates real-time indicators of compromise (IoCs) and exports blacklists to system administrators, firewalls, and browser extension maintainers worldwide, completely free of charge.
